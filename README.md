# Smart_Home
### Arduino Mega Projesi
Projede Proteus programında Arduino kartı kullanarak akıllı ev sistemi oluşturulmaktadır. Sistem içerisinde;
• Yangın alarmı,
• Hareket algılayan ışık sistemi,
• Dijital termometre,
• Kilit sistemi bulunmaktadır.
